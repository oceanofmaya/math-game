# Sound System

The sound system provides theme music for each visual theme, creating an immersive and calming atmosphere that matches the game's aesthetic.

## Music Tracks

The theme music tracks are sourced from Kevin MacLeod's library at [Incompetech](https://incompetech.com/), all licensed under Creative Commons: By Attribution 4.0 License.

### Current Music Tracks

- **Jellyfish Theme**: "Ambiment" by Kevin MacLeod
- **Firefly Theme**: "Ethereal Relaxation" by Kevin MacLeod
- **Pearl Theme**: "Kalimba Relaxation Music" by Kevin MacLeod
- **Mushroom Theme**: "Magic Forest" by Kevin MacLeod
- **Maple Theme**: "Evening" by Kevin MacLeod

All tracks are licensed under [Creative Commons: By Attribution 4.0 License](http://creativecommons.org/licenses/by/4.0/).

## Usage

### How It Works

- Music files are located in `public/sounds/music/`
- Music configuration is in `public/js/sound-manager.js`
- Each theme has its own unique music track
- Music plays at 30% volume by default
- Music loops seamlessly when a theme is selected
